# Blog API
This project is a RESTful API built with Flask for managing users, posts, comments, categories, and favorites.

## Tech Stack
- Python 3
- Flask
- SQLAlchemy
- Flask-JWT-Extended
- Flask-Caching
- Swagger (Flasgger)

## Installation

```bash
pip install -r requirements.txt
```
## Run the Application

```bash
python seed.py
python app.py
```

## Run tests 
```bash
pytest -v
```
